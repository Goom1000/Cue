# Requirements: Cue v3.4 Ask AI

**Defined:** 2026-01-26
**Core Value:** Teachers can ask AI anything during a presentation without leaving the app

## v3.4 Requirements

Requirements for the Ask AI in-presentation assistant. Teachers ask questions, get AI answers using lesson context.

### Core Input/Output

- [x] **CHAT-01**: Teacher can type a question in a text input field in the teleprompter panel
- [x] **CHAT-02**: AI response streams character-by-character as it generates (not chunky blocks)
- [x] **CHAT-03**: Loading indicator shows "Thinking..." while waiting for AI response
- [x] **CHAT-04**: Teacher can copy AI response to clipboard with one click
- [x] **CHAT-05**: Errors display friendly messages with retry option

### Context & Intelligence

- [x] **CTXT-01**: AI receives current slide content, title, and cumulative lesson context
- [x] **CTXT-02**: AI response uses age-appropriate language matching the lesson's grade level
- [x] **CTXT-03**: Quick action buttons provide preset prompts ("Get 3 facts", "Explain simply", "Answer student question")

### UX & Safety

- [x] **UX-01**: Assistant panel is inline in teleprompter (not modal, never steals focus)
- [x] **UX-02**: Assistant is teacher-only (not synced to student view via BroadcastChannel)
- [x] **UX-03**: Privacy indicator shows "Not visible to students" in the panel

### Session History

- [x] **HIST-01**: Q&A history persists during the presentation session
- [x] **HIST-02**: Teacher can view previous questions and answers in scrollable list
- [x] **HIST-03**: Teacher can clear session history

### Keyboard Access

- [x] **KEY-01**: Keyboard shortcut (Cmd/Ctrl+K) focuses the chat input
- [x] **KEY-02**: Escape key returns focus to presentation navigation
- [x] **KEY-03**: Arrow keys always navigate slides (never captured by chat input)

## Deferred (v3.5+)

- Voice input via microphone
- Suggested queries based on slide content
- Cross-session conversation persistence
- Token/cost counter in settings

## Out of Scope

| Feature | Reason |
|---------|--------|
| Voice input | Classroom noise, complexity â€” defer to future |

| Full conversation persistence | Scope creep, session-only is sufficient |
